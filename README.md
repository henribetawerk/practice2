# A Kanban Maybe

This project is a Kanban-style content management board for Week 2.

## Features

### Current Implementation
- **Visual Kanban Board**: 6-column layout (New, Planning, Active, Revising, Overdue, Closure)
- **Content Creation**: Add new content ideas through input form
- **Responsive Design**: Clean, modern UI with soft color palette
- **State Management**: Track content items across different workflow stages

### Technical Stack
- Pure HTML/CSS/JavaScript (ES6 modules)
- Local state management
- Modular architecture with separate data layer

### Workflow Columns
1. **New** - Fresh content ideas
2. **Planning** - Ideas being developed
3. **Active** - Content in production
4. **Revising** - Content under review/editing
5. **Overdue** - Delayed content items
6. **Closure** - Completed content

## Goals
- Practice Spec-driven development
- Learn how to manage Context when state is introduced
- Build functional Kanban board interface
